# JT Smart Modal Pro Updates

This repository hosts the public update metadata for JT Smart Modal Pro.

## Contents

- `updates.xml` — Joomla update server definition
- `LICENSE` — repository license information

## Notes

JT Smart Modal Pro is a commercial Joomla extension.

This repository does not contain the distributable extension source code or package.

The update definition points to a protected download endpoint managed by JoomTheme.
